## 시각화

+ ### 선(연속적)＆박스
  ```angular2html
  테이블_변수['속성'].plot(kind='line/box')
  ```

+ ### 히스토그램＆밀도(가로: 분포, 세로: 빈도)
  ```angular2html
  테이블_변수['속성'].plot(kind='hist/kde')
  ```

+ ### 산점도(점)
  ```angular2html
  테이블_변수.plot(x='x축_이름', y='y축_이름', kind='scatter')
  ```

+ ### 막대그래프(속성별)
  ```angular2html
  테이블_변수.groupby( ['속성_리스트'] )['속성'].통계()
      .plot(kind='bar/barh')  # 가로/세로
  ```

+ ### 파이차트(속성별)
  ```angular2html
  테이블_변수.groupby('속성')['속성'].통계()
      .plot(kind='pie')
  ```
